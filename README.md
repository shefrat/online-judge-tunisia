# online-judge-tunisia

A modern open-source online judge and contest platform system, built on DMOJ: Modern Online Judge. It will used to host competitions including national olympiads in Tunisia.

## Supported Languages
Supported languages include:
* C++ 11/14/17 (GCC and Clang)
* C 99/11
* Java 8/9/10/11
* Python 2/3
* PyPy 2/3
* Pascal
* Perl
* Mono C#/F#/VB
